# Flash DB

Flash DB is an async ORM for Python, built on SQLAlchemy. It offers a Django-style API for modern async applications, especially those using FastAPI.

## Core Features

- **Async Native:** Built for `asyncio`.
- **Django-like API:** A familiar `Model.objects` manager and lazy `QuerySet`s.
- **SQLAlchemy Core:** The full power of SQLAlchemy's expression language when you need it.
- **Type-Safe:** Fully type-annotated for better code quality.
- **Minimal Setup:** Easy to integrate into your project.
